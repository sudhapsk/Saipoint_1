Section III Using IdentityIQ : IdentityIQ Dashboards : Compliance Dashboard Components : Certifications
Certifications
A list view of certifications that includes the name of the certification, the owner, completion status and the date the certification was created.
Click a certification to display additional information about the certification including links to the certification schedule and the access review. See Access Review Page Overview.
Use the search features to limit the number of access reviews displayed.
