Forward an Approval
Use the following steps to forward an approval.
1.
Select Forward from the Actions menu for an approval.
2.
Enter the name or a few letters of the name of the new owner of the approval. Alternatively, you can click the down icon and select a name from the list.
3.
Add any forwarding comments and click Forward.
