Manage User Access Page
IdentityIQ can be set up to request and manage access for single or multiple identities. Based on how your system is configured, you can manage:
•
Access for Multiple Users — Users request and manage access for one or more identities. This option can also be set up to enable you to request access for yourself.
•
Access for a Single User — Users request and manage access for their own access.
Note:
If you click the Home button, exit the IdentityIQ application, or navigate away from the manage access pages before you complete all tasks, your entries are cleared and the access request is NOT submitted.
