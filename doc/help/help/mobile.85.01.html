IdentityIQ on Mobile Devices
To access the IdentityIQ Mobile Login page, use a supported mobile browser and enter
http://<iiq server>/IdentityIQ/ui/. For a list of supported browsers and mobile platforms, see the IdentityIQ Release Notes. The mobile IdentityIQ platform supports direct links that allow users to create and use direct links into IdentityIQ pages from outside of the product from places such as emails, forms, or portal. For more information, see the Direct Links information in the IdentityIQ Administration Guide.
